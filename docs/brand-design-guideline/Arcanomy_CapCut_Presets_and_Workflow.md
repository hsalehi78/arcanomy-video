# Arcanomy Motion — CapCut Setup & Presets v1.0
_Last updated: 2025-12-30_

This is the fastest way to produce daily reels using the Template Pack.

---

## 1) Create a CapCut Project Template
Make one master project called:
**`Arcanomy_Reel_Master_9x16_v1`**

### Tracks
- V1: Background video / b-roll
- V2: Charts / screenshots / overlays
- V3: Text (TYPE_XL/L/M)
- V4: Labels (TYPE_S), “RECEIPT”, margin notes
- A1: Voiceover
- A2: Music bed
- A3: SFX

---

## 2) Save Text Styles (must-do)
Create and save these as presets:
- TYPE_XL
- TYPE_L
- TYPE_M
- TYPE_S

Rules:
- Never invent a new style mid-reel.
- Max 2 lines.
- Prefer fewer words.

---

## 3) Save Animation Presets (must-do)
Create 3 animation/keyframe presets:
- REVEAL_DEFAULT (opacity + y)
- MARKUP_DRAW (stroke draw-on)
- NUMBER_FREEZE (brief pause + big number)

If CapCut limits easing controls, you still enforce the **feel**:
- quick start, soft stop
- no bounce

---

## 4) Build 5 Template Timelines (once)
Create five CapCut projects and keep them as duplicates:
- Template A — The Receipt
- Template B — Two Timelines
- Template C — Hidden System
- Template D — 60s Audit
- Template E — The Rule

Each should contain:
- placeholder text blocks labeled by TYPE
- placeholder “RECEIPT” label
- placeholder chart layer slot
- placeholder b-roll slots
- a “safe zone” guide overlay (optional)

---

## 5) Daily Production Workflow (10-step)
1) Pick template (A–E) based on the claim.
2) Paste script into VO track (or record).
3) Place hook text (TYPE_XL) + underline motif.
4) Drop 2–3 b-roll clips (supporting meaning).
5) Create receipt moment (chart/screenshot) + markup.
6) Freeze on one number (if applicable).
7) Add the rule line (TYPE_L) + support (TYPE_M).
8) Add minimal SFX (marker/tick only where needed).
9) Quick QA (word count per screen, calm pacing).
10) Export using preset.

---

## 6) “Don’t waste time” rules
- If a reel needs more than 1 chart, it’s two reels.
- If you’re tweaking colors, you’re procrastinating. Use tokens.
- If type is moving a lot, it’s not Arcanomy.
